
JTEN Final Complete Frontend
----------------------------
How to run:
1. Extract the ZIP.
2. Open index.html in Chrome or Edge.
3. Login as admin / 123
4. Admin -> User Access Management to create Registrar and Professor accounts.
5. Create Rooms and Subjects.
6. Registrar -> Registration: use Tap card input to capture UID (USB RFID readers emulate keyboard input) or type UID then Enter; fill name/password to create student.
7. Assign subjects, create schedules, assign seats via Registrar.
8. Professors: use Professor page to Tap In/Out and approve/reject LOA requests.
9. Students: login with Student No (UID) and password; they land on their Student page (attendance, LOA, schedule).
Notes:
- Data stored in browser localStorage under key 'jten_final_complete_data'.
- Export attendance from Reports page.
